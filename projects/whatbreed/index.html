                  WhatBreed

                  Published: 2019-10-01

                    Updated: 2020-02-10

                  # Summary

                  WhatBreed is a simple tool for detecting the breed of your pet using pictures which can be found at https://whatbreed.ai.

                  # Data

                  The data is put together from many sources across the Internet, including the Oxford Pet Dataset, Stanford Dogs Dataset as well as free image sites such as Wikimedia and Pexels, etc. Between the various datasets, this includes the 100 most popular dog breeds identified by the American Kennel Club and the 25 most popular cat breeds identified by the Cat Fanciers Association.

                  # Architecture

                  The model was built using fastai using resnet50 convolutional neural network. No significant feature engineering was done beyond using get_transforms() from the fastai library.

                  # Application

                  The backend is a Dockerized web server built on Python using Starlette, which accepts up to nine pictures, forwards them to the fastai learner to predict the breeds. The backend is hosted on Render.

                  The frontend is a progressive web app built using VueJS and Axios that uses HTML input to open folders or access the camera to take pictures to send to the server. Being a PWA, the application is installable through Chrome, which is especially useful on smartphones.

                  # Ethical and Safety Considerations

                  This application is intended for educational and discovery use. It has not been tested in a medical setting and should not be used as a basis for making medical diagnosis. Please talk to a qualified veterinarian regarding breed related health concerns.

                  # Future Plans

                  • make available to local shelters
                  • donate application to PetFinder (currently in talks)
                  • coordinate with veterinary professionals on qualifying the tool for medical diagnostic usage in the future
